# Neuroimaging Research Notes

Personal research notes on neuroimaging analysis, paper readings, and generative models for brain imaging.

## Repository Structure

```
neuroimaging-research-notes/
├── paper-reading-notes/
│   ├── structural-mri/        # Cortical thickness, volumetrics, morphometry
│   ├── functional-mri/        # fMRI, resting-state, functional connectivity
│   ├── diffusion-mri/         # DWI, DTI, tractography
│   └── deep-learning-neuroimaging/  # DL-based segmentation, classification, etc.
├── neuroimaging-tools-notes/  # FreeSurfer, FSL, ANTs, nilearn, etc.
└── generative-model-notes/    # Diffusion models, VAE, GAN, flow matching
```

## Topics Covered

### Paper Reading Notes
- **Structural MRI**: FreeSurfer-based cortical analysis, voxel-based morphometry (VBM), surface-based morphometry (SBM)
- **Functional MRI**: Task-based fMRI, resting-state functional connectivity, independent component analysis (ICA)
- **Diffusion MRI**: Diffusion tensor imaging (DTI), constrained spherical deconvolution (CSD), white matter tractography
- **Deep Learning for Neuroimaging**: Brain age prediction, lesion segmentation, disease classification

### Neuroimaging Tools
- FreeSurfer, FSL, ANTs, SPM, nilearn, MRtrix3, and more

### Generative Models
- Diffusion models (DDPM, score-based models)
- Flow matching and continuous normalizing flows
- VAE and GAN architectures
- Applications in medical image synthesis, data augmentation, and anomaly detection

## About

These notes are part of my ongoing study in computational neuroimaging and machine learning for brain imaging research.
